## 修改职位所拥有的权限



请求路径：`/api/permission_role/4/permissions/`

请求方式： `post`

请求参数： 

| 参数      | 数据类型 | 数据范围 | 必传 | 备注                                             |
| --------- | -------- | -------- | ---- | ------------------------------------------------ |
| permission_ids | array      |          | 是   | 权限id列表                     |


请求示例：

```
{
    "permission_ids": [1,2,3]
}
```



返回参数说明：

| 参数    | 数据类型 | 备注说明               |
| ------- | -------- | ---------------------- |
| code    | int      | 200 为成功，其他为异常 |
| message | string   | 提示信息               |
| data    | array    | 返回数据               |

响应示例：

```json
{
    "msg": "",
    "code": 200,
    "data": {
    }
}
```
